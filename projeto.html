SuDoKu BR

Este projeto começou com uma simples dúvida: como fazer um algoritmo para gerar tabuleiros aleatórios de SuDoKu?

A partir daí utilizei Go para desenvolver um caminho lógico para a geração de linhas, colunas e quadrantes válidos.

Foram várias tentativas:

  • Linha após linha;
  • Coluna após coluna;
  • Uma linha, uma coluna, etc;
  • Um quadrante por vez;
  • Número a número;

Ao rodar os algoritmos, logo na terceira ou quarta linha o computador já não encontrava lugares válidos para novos números, e a cada nova ideia o sonho de gerar rapidamente resoluções de jogos parecia mais próximo, mas novas frustrações apareciam :(

A primeira vez que consegui imprimir uma matrix 9x9 válida considerando os parâmetros do Sudoku foi limpando o algoritmo e tentado do começo novamente, algumas vezes o número de tentativas passa de 10.000, outras vezes menos de 500. Sigo na busca por algoritmos mais eficientes!
